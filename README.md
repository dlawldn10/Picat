# Picat
## Sw정글 나만의 무기 프로젝트
얼굴 인식으로 쉽고 빠른 사진 공유 안드로이드 앱 Picat

## 맡은 역할
- 안드로이드 스튜디오 & 코틀린 클라이언트 개발
- UX/UI 디자인 및 레이아웃 구성
- 사진 선택 인터랙션
- retrofit api 연결
- socket.io 연결
